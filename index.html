Aplikasi Desktop / Network Operations Tool

EDP NetOps

EDP NetOps adalah aplikasi monitoring dan manajemen operasional IT berbasis Flutter yang dibuat untuk membantu kebutuhan IT Support dan Network Operations. Aplikasi ini digunakan untuk mengelola data jaringan toko/cabang, memantau kondisi perangkat, mencatat riwayat tiket gangguan, serta menyediakan beberapa tools jaringan untuk mendukung proses troubleshooting secara lebih terstruktur.

Tujuan Aplikasi

Aplikasi ini dibuat untuk membantu proses monitoring jaringan dan pengelolaan data teknis toko/cabang agar pekerjaan IT Support menjadi lebih cepat, rapi, dan terpusat. Dengan adanya dashboard, data perangkat, riwayat tiket, dan tools jaringan, proses pengecekan gangguan dapat dilakukan dengan lebih efisien.

Fitur Utama

  • Dashboard monitoring dan ringkasan kondisi jaringan
  • Pengelolaan data toko/cabang secara terpusat
  • Detail perangkat jaringan seperti router, gateway, kasir, STB, WDCP, dan CCTV
  • Pencatatan dan pengelolaan history ticket gangguan
  • Analisis atau ranking gangguan provider/ISP
  • Ping scanner untuk membantu pengecekan koneksi perangkat
  • Integrasi tools jaringan seperti MikroTik API dan remote tools
  • Export data atau laporan ke file Excel
  • Dukungan dark mode dan light mode
  • Role-based access untuk membatasi akses berdasarkan peran pengguna

Teknologi

Flutter Dart Supabase Node.js TypeScript IMAP Worker SMTP Email MikroTik API Excel Export

Network Automation

Bot Tambah MAC

Bot Tambah MAC adalah aplikasi desktop Windows berbasis Flutter yang berfungsi sebagai server penghubung antara Telegram Bot, Supabase, dan router MikroTik. Aplikasi ini membantu proses pengelolaan MAC Address pada wireless access-list MikroTik secara jarak jauh melalui perintah Telegram, sehingga pekerjaan teknis dapat dilakukan lebih cepat dan terstruktur.

Tujuan Aplikasi

Mempermudah tim IT Support atau administrator jaringan dalam melakukan registrasi MAC Address, mengecek perangkat wireless yang sedang terhubung, memantau status router MikroTik, dan melakukan restart router dari jarak jauh tanpa harus membuka Winbox atau melakukan akses manual secara langsung.

Fitur Utama

  • Registrasi MAC Address jarak jauh via /tambah
  • Validasi format MAC & pengecekan duplikasi otomatis
  • Monitoring perangkat wireless aktif via /cek
  • Pengecekan status router via /status
  • Restart router jarak jauh via /restart
  • Sistem antrean request agar koneksi tidak bentrok
  • Cache konfigurasi MikroTik dari Supabase
  • Panel desktop monitoring status, uptime, log & error
  • Integrasi system tray Windows untuk background

Teknologi

Flutter Desktop Dart Supabase Telegram Bot API MikroTik RouterOS API TCP Socket
