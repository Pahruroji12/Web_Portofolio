# Portofolio Pahruroji

Website portofolio pribadi Pahruroji yang menampilkan profil, keahlian, pengalaman, project, dan informasi kontak. Website dibuat responsif agar tetap nyaman digunakan melalui desktop maupun perangkat mobile.

## Teknologi

- HTML
- CSS
- JavaScript
- HTML5 Canvas
- Boxicons dan Devicon

## Fitur Utama

- Dark mode dan light mode
- Background canvas interaktif untuk desktop
- Preloader dan animasi saat halaman ditampilkan
- Daftar project dengan gallery, video, dan modal detail
- Live preview project khusus desktop
- Tampilan responsif untuk desktop dan mobile
- Form kontak demo serta tautan email dan WhatsApp
